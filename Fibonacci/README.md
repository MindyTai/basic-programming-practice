Fibonacci
==
- use 「if-else」to deal with input 0 and input 1，
-  then use 「for」 to deal with other inputs.

```
input  numA   numB   res
0   -> x      x      dont care (0)
1   -> x      x      dont care (1)
2   -> 0      1      1
3   -> 1      1      2
4   -> 1      2      3
5   -> 2      3      5

F3 = F2 + F1
   = F1 + F0


Fn = F(n-1) + F(n-2)
F(n+1) = F(n) + F(n-1)
```
